# Sundaram Finance — AI-Enhanced CRM Platform

An integrated CRM and Loan Origination System for housing finance, built with React + Vite.

## Modules

| Module | Description |
|--------|-------------|
| Dashboard | Live pipeline funnel, AI alerts, team performance |
| Lead Management | Omnichannel capture — app, web, chatbot, branch, partners |
| Customer 360° | Unified profile, AI OCR documents, interaction timeline |
| Loan Origination (LOS) | 6-step workflow: Application → KYC → AI Risk → Underwriting → Approval → Disbursement |
| Property Appreciation AI | Scenario forecasting (base/optimistic/stress) over loan tenure |
| DPR Viability Engine | OCR + NLP extraction and project viability scoring for indirect finance |
| AI Risk Scoring | Pre-sanction + post-disbursement dynamic risk monitoring |
| Collections & Service | DPD buckets, overdue EMI management, service tickets |
| AI Financial Advisor | Conversational chatbot for EMI, rates, property comparison |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
```

Deploy the `dist/` folder to Vercel, Netlify, or any static host.

## Tech Stack

- React 18
- Vite 5
- Pure CSS (no UI library)
